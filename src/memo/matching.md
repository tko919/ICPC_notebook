## Hallの定理
頂点が $U,V$ に分割された二部グラフ上で、「 $U$ をカバーするマッチングが存在」$\iff$「 $\forall A \subset U,|A| \leq |\delta(A)|$ 」

## 二部グラフのマッチング関連
|  | サイズ | 構成 |
| ---- | ---- | ---- | 
| 最小点被覆 | $M$ | $V\setminus M$ から $L$ 到達不可能+$R$ 到達可能 | 
| 独立集合 | $V-M$ |  上の補集合  |
| 最小辺被覆 | 孤立点ないなら $V-M$ | $M$ +入らない頂点でGreedy | 

## Dilworthの定理
推移的なDAG上で、最小path被覆のサイズと最大独立集合のサイズは同じ。
c.f. (DAG上の最小path被覆のサイズ)= $N$ -(頂点倍化した二部グラフの最大マッチング)が成り立つ。
